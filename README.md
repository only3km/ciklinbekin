# 平話字表 (閩東語‣福州話) 

`🚧 校對中, 尚未完善, 請謹慎取用｡`

爲便於查詢平話字音､ 使平話音輸入漢字, 故取讀音字表將其電子化｡  
同時生成HTML網頁和Rime輸入法碼表｡

## 來源
- 戚林八音校注, 2001, 蔡士泮, 陳他, 李如龍, 王升魁(著) 的 字
- Dictionary of the Foochow Dialect, Third Edition, 1929 的 Index of Characters
